# Data Communication
Moving data from one part to another relies on *PDU* or *Protocol Data* Unit

## PDU
- Sometimes mention as *Transmission units* 
	- Is a different groups of data at different OSI layer

Example:
An Ethernet will send everything in a #Ethernet_Frame to one device from another MAC address devices in the network. It doesn't care what is in the inside

Similarly to the [[Network]] Layer, IP operates on packet of data wether it is a TCP or a UDP, it doesn't really care
