
#network_interface_card 

is a hardware that lets a computer connect to a network. Implements circuitry in the physical layer to communicate with the data link layer

#MAC_address 

is a hardware identification number that identifies all devices that is connected to the network.

#Frames 

A container for a single network packet complete with addressing and necessary protocol control informations

#SSL

Secure Socket Layer, a technology that keeps an internet connection secure and protecting data being sent over between two devices

#Ethernet_Frame 

An Ethernet Frame contains and starts with the header, the source and destination MAC address and in the middle part contains the actual data and ends with a field called Frame Check Sequence where the receiving system is informed to enable syncronisation
